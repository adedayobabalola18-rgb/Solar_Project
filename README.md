# Solar Project

A static solar services website composed of several page sections and a shared JavaScript file for interactive behavior.

## Project Overview

This project is a small solar energy landing site with pages for:
- Landing / Home
- About
- Services
- Products
- Contact

It includes a UI for estimating savings, theme toggling, a mobile sidebar, and a contact form.

## Folder Structure

- `Landing_page/` - Home page markup and styling
- `About_page/` - About page markup and styling
- `Services_page/` - Service details page markup and styling
- `Product_page/` - Product page markup and styling
- `Contact_page/` - Contact page markup and styling
- `js/` - Shared JavaScript functionality

## Key Files

- `Landing_page/Solar.html` - Home page HTML
- `Landing_page/Solar.css` - Home page styles
- `About_page/solar-about.html` - About page HTML
- `About_page/solar-about.css` - About page styles
- `Services_page/solar-services.html` - Services page HTML
- `Services_page/solar-services.css` - Services page styles
- `Product_page/solar-products.html` - Product page HTML
- `Product_page/solar-products.css` - Product page styles
- `Contact_page/solar-contact.html` - Contact page HTML
- `Contact_page/solar-contact.css` - Contact page styles
- `js/Solar.js` - JavaScript for theme toggling, savings slider, scroll behavior, sidebar, and contact form handling

## How to Use

1. Open any of the `.html` files in your browser.
2. The landing page can be opened via `Landing_page/Solar.html`.
3. The site uses Font Awesome for icons via CDN.

## Notes

- The site is currently a static frontend project and does not require a build step.
- Some file links may be set to local absolute paths and might need updating to relative paths for a portable deployment.
- To view the pages consistently, open them in a browser or use a local static server.
